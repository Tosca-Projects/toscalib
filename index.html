    • Toscalib

    A go implementation of the TOSCA Yaml specifications

    Project maintained by owulveryck Hosted on GitHub Pages — Theme by mattgraham

    Abstract

    This library is an implementation of the TOSCA definition as described in the document written in pure GO TOSCA Simple Profile in YAML Version 1.0

    Normative Types

    The normative types definitions are included de facto. The files are embeded using go-bindata.

    Howto

    Create a ServiceTemplateDefinition and call Parse(r io.Reader) to fill it with a YAML definition.

    Example

    var t toscalib.ServiceTemplateDefinition
err := t.Parse(os.Stdin)
if err != nil {
    log.Fatal(err)
}
